### メイツとは？
Web プラットフォームのアプリケーションを主軸とした EdTech 企業です！\
学習塾向け ICT 教材 [aim@](https://mates-app.jp/aim/) や、塾管理システム **reco** を始め、ほぼ全てのプロダクトで Vue.js / Nuxt を使っています！ 
Vue.js はもちろん AWS 等でコミュニティ活動や OSS 活動をしているメンバーも多数在籍しており、チーム全体で OSS への参加が前向きな組織です！！
フロントエンドエンジニアも、バックエンドエンジニアも、PM も絶賛大募集中！！
